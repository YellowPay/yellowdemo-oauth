This is a demo application for Yellow payments. It demonstrates how a 3rd party application can request authorization, via OAuth, to create and monitor invoices on behalf of a merchant.

No OAuth credentials will be stored on the server so you will need to grant the application access each time you use the site.

The full source of the application can be found on GitHub.
